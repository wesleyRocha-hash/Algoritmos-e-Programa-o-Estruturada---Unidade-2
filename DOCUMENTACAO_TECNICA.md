# Relatório Técnico: Estruturas de Decisão em C

## 1. Fundamentação Teórica
Nesta unidade, exploramos como o fluxo de um programa deixa de ser linear e passa a ser ramificado através de:
* **Estrutura Simples (`if`)**: Executa comandos apenas se a condição for verdadeira.
* **Estrutura Composta (`if-else`)**: Define um caminho alternativo caso a condição falhe.
* **Estrutura de Seleção (`switch-case`)**: Ideal para menus, testando uma variável contra constantes inteiras ou caracteres.

## 2. Análise Técnica (Depuração)
Durante o desenvolvimento no **Google Colab**, identificamos a importância da ordem das condições. Por exemplo, ao usar $num \geq 0$, o zero é incluído no "Positivo". Para maior precisão técnica, utilizamos o encadeamento `else if` para isolar o zero como um caso neutro.
